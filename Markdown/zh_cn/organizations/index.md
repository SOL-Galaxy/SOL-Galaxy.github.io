# 势力组织

## 欧加斯财团 (The Ogas Consortium)
故事世界的实际统治者。一个致力于秘密合法化非法谋取暴利的bt组织，数十年来隐瞒得很好，几乎没有人知道在光的背面，他们干着如此龌龊不堪的事。旗下有多家实验室。其中最为人所知的就是拓扑实验室。财团控制着两颗星球上所有的基础生存资源——包括**水源净化**、能源分配、星际航道，乃至空气处理系统。
